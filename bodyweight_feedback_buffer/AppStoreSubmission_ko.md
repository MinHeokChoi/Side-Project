## 2. App Store 제품 페이지

### 홍보 문구

단순히 횟수와 세트 수만 기록하고 계신가요?
맨몸 운동은 동작의 '질'과 '감각'이 무엇보다 중요합니다.
피드백 버퍼는 당신이 운동 직후 느낀 미세한 감각과 개선점을 잊지 않도록 도와주는 전용 기록 앱입니다.

### 설명

맨몸운동의 성장은 숫자만으로 설명되지 않습니다.

몇 세트를 했는지, 몇 회를 성공했는지도 중요하지만 Handstand에서 어깨가 살짝 말린 순간, Pull ups에서 반동이 들어간 느낌, Front Lever에서 코어가 풀린 감각처럼 다음 훈련을 바꾸는 단서는 더 작고 빠르게 사라집니다.

피드백 버퍼는 운동 직후의 미세한 감각, 코칭 포인트, 다시 확인해야 할 개선점을 기술별로 모아두는 개인 훈련 기록 앱입니다. 잊기 쉬운 피드백을 짧게 남기고, 다음 훈련 전에 다시 꺼내보세요.

Handstand, HSPU, Pull ups, Front Lever, Dips, Muscle Up처럼 꾸준히 다듬어야 하는 기술을 나누어 관리할 수 있습니다. 기록한 피드백은 중요도, 오래 방치된 정도, 더 연습이 필요했던 횟수를 바탕으로 정렬되어 오늘 먼저 집중할 포인트를 보여줍니다.

주요 기능

- 기술별 피드백 기록: 기술마다 다른 감각, 실수, 코칭 포인트를 분리해 기록
- 피드백 버퍼: 아직 해결하지 못한 피드백을 우선순위대로 확인
- 해결/더 연습 표시: 반영된 피드백은 해결하고, 다시 놓친 피드백은 더 연습으로 표시
- 빠른 문구 관리: 자주 쓰는 피드백 문구를 직접 추가, 삭제, 순서 변경
- 기술 라이브러리: 내 훈련 기술을 추가, 수정, 삭제, 순서 변경
- 웜업 체크리스트: 매일 확인하는 웜업 루틴과 직접 편집 가능한 항목
- 기기 내 저장: 계정 가입 없이 기록을 기기에 저장

피드백 버퍼는 운동을 대신 분석해주는 앱이 아니라, 당신이 직접 느낀 중요한 단서를 놓치지 않게 붙잡아두는 앱입니다. 다음 훈련에서 같은 실수를 줄이고, 오늘의 감각을 내일의 개선으로 이어가세요.

이 앱은 의학적 진단, 치료, 재활 처방을 제공하지 않습니다. 운동 중 통증이나 부상이 있다면 전문가의 도움을 받아주세요.

## 3. 스크린샷 및 앱 미리보기

필수: iPhone용 스크린샷 1-10장, iPad용 스크린샷 1-10장. 현재 프로젝트가 iPhone과 iPad를 모두 지원하므로 iPad 스크린샷도 준비해야 합니다.

권장 스크린샷 구성:

1. 피드백 버퍼 탭: 우선순위가 높은 피드백 카드
2. 기술 라이브러리 탭: 기술별 피드백 개수와 새 아이콘
3. 기술 상세 화면: 특정 기술에 쌓인 피드백
4. 새 피드백 작성 화면: 빈 공간, Cue, 중요도, 빠른 문구
5. 빠른 문구 관리 또는 기술 관리 화면
6. 웜업 탭: 진행률과 오늘의 웜업 체크리스트
7. 웜업 루틴 수정 화면

앱 미리보기 영상: 선택 사항. 첫 제출에서는 생략해도 됩니다.

### 심사 메모

피드백 버퍼는 로그인 없이 바로 사용할 수 있는 개인 훈련 피드백 기록 앱입니다. 첫 실행 시 온보딩에서 "시작하기"를 누르면 기본 기술 목록과 기본 웜업 루틴이 표시됩니다.

테스트 방법:

1. 온보딩에서 "시작하기"를 누릅니다.
2. 기술 라이브러리 탭에서 기술을 선택하고 + 버튼으로 새 피드백을 추가합니다.
3. 피드백 버퍼 탭에서 해결, 더 연습, 수정, 삭제 동작을 확인합니다.
4. 새 피드백 화면의 빠른 문구 편집 버튼으로 문구 추가, 삭제, 순서 변경을 확인합니다.
5. 기술 라이브러리의 관리 버튼에서 기술 추가, 수정, 삭제, 순서 변경을 확인합니다.
6. 웜업 탭에서 체크리스트를 체크하고, 우측 상단 편집 버튼으로 루틴을 수정합니다.

네트워크 통신, 계정 생성, 인앱 결제, 광고, 분석, 추적 기능은 없습니다. 사용자가 입력한 기술, 피드백, 메모, 빠른 문구, 웜업 정보는 기기 내에만 저장됩니다. 별도 테스트 계정은 필요하지 않습니다.
