# Changelog

Feedback Buffer의 배포별 변경 내역을 기록합니다.

## 1.0.1 - 예정

### Added
- 기술 라이브러리의 기술 상세 화면에 보관함 탭이 추가됐어요. 보관한 피드백을 다시 꺼내 연습할 수 있어요.
- 기술 라이브러리 카드에 보관된 피드백 개수도 함께 표시돼요. 활성은 강조색, 보관은 초록색으로 구분돼요.
- 웜업 세션을 여러 개 만들고 전환할 수 있어요. 세션 이름과 항목 목록만 입력하면 한 번에 만들어져요.

### Changed
- 피드백 버퍼의 범주 필터를 "전체 / 체력 훈련 / 기술 훈련" 단일 선택으로 바꿨어요.
- 카드의 "미해결" 버튼을 "연습했어요"로 바꿨어요. 누르면 연습 횟수가 1 늘어나고 마지막 연습일이 오늘로 갱신돼요.
- 카드의 "해결"은 "보관"으로 바꿨어요. 보관한 피드백은 기술 상세의 보관함에서 다시 볼 수 있어요.
- 피드백이 새 피드백 → 연습 시작 → 적응 중 → 보관 흐름을 따라가도록 정리했어요.
