# Changelog

Feedback Buffer의 배포별 변경 내역을 기록합니다.

## 1.1 - 예정

### Added
- 기술 라이브러리에 검색을 추가했어요. 기술 이름으로 빠르게 찾을 수 있어요.
- 기술 라이브러리의 기술 상세 화면에 보관함 탭이 추가됐어요. 보관한 피드백을 다시 꺼내 연습할 수 있어요.
- 기술 라이브러리 카드에 보관된 피드백 개수도 함께 표시돼요. 활성은 강조색, 보관은 초록색으로 구분돼요.
- 웜업 세션을 여러 개 만들고 전환할 수 있어요. 세션 이름과 항목 목록만 입력하면 한 번에 만들어져요.
- 웜업 화면에 "오늘의 웜업 시작" 버튼이 추가됐어요. 항목을 한 장씩 카드로 보여주며 따라갈 수 있어요.

### Fixed
- 피드백 카드 우선순위 점수 정보 팝오버의 텍스트가 잘려 보이던 문제를 수정했어요.

### Changed
- 웜업 화면의 큰 타이틀이 현재 세션 이름을 직접 보여줘요. 세션 변경 행은 항목 수만 표시해 중복 없이 깔끔해졌어요.
- 기술 카드 하단에 마지막 연습 시점이 표시돼요. 얼마나 오래됐는지 한눈에 알 수 있어요.
- 피드백이 비어 있을 때 기술 라이브러리로 바로 이동하는 버튼이 생겼어요.
- 피드백 점수 옆 아이콘을 탭하면 점수 산정 방식과 단계별 색상 의미를 확인할 수 있어요.
- 피드백 버퍼의 범주 필터를 "전체 / 체력 훈련 / 기술 훈련" 단일 선택으로 바꿨어요.
- 카드의 "미해결" 버튼을 "연습했어요"로 바꿨어요. 누르면 연습 횟수가 1 늘어나고 마지막 연습일이 오늘로 갱신돼요.
- 기술 카드의 활성/보관 카운트 옆에 작은 아이콘이 붙었어요. 색상이 잘 안 보이는 환경에서도 어떤 상태인지 바로 알 수 있어요.
- 피드백이 새 피드백 → 연습 시작 → 적응 중 → 보관 흐름을 따라가도록 정리했어요.
- 웜업 세션 목록의 이름 변경/항목 수정/삭제를 행 우측 점 세 개 메뉴 한 곳에서 할 수 있어요.
