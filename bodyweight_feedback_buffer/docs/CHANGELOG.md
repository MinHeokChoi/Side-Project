# Changelog

Feedback Buffer의 배포별 변경 내역을 기록합니다.

## 1.0.1 - 예정

### Changed
- 피드백 버퍼의 범주 필터를 "전체 / 체력 훈련 / 기술 훈련" 단일 선택으로 바꿨어요.
- 카드의 "미해결" 버튼을 "연습했어요"로 바꿨어요. 누르면 연습 횟수가 1 늘어나고 마지막 연습일이 오늘로 갱신돼요.
- 카드의 "해결"은 "보관"으로 바꿨어요. 보관한 피드백은 기술 상세의 보관함에서 다시 볼 수 있어요.
